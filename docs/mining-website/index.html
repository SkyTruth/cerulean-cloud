Public API

SkyTruth Mining API Guide

SkyTruth publishes an OGC API Features-compatible TiPG endpoint at https://api.mining.skytruth.org/ for mining footprints, permit datasets, highwall detections, and supporting reference layers. This page translates the existing Mining API guide script into a structured reference modeled on the same approach as the Cerulean docs site.

Base URL

https://api.mining.skytruth.org/

Main pattern

GET /collections/{collectionId}/items

Coverage

SkyTruth mining + permit + reference layers

For bulk downloads of MTM datasets, use MTM Data Files.

For API semantics and filter syntax, see TiPG and CQL-2.

Mine footprints

Annual mining footprints from 1985-2025, with `final` and `provisional` status.

public.annual_mining SkyTruth

Permit datasets

State permit tables for Kentucky, Tennessee, Virginia, and West Virginia.

public.state_permits_ky public.state_permits_tn public.state_permits_va public.state_permits_wv

Reference layers

Highwall detections, e-AMLIS, counties, and HUC boundaries for context and joins.

Quickstart

Start with a bounding-box query against public.annual_mining. The sample coordinates come directly from the existing guide. 

curl --globoff \
  'https://api.mining.skytruth.org/collections/public.annual_mining/items?bbox=-81.8156,37.7981,-81.7882,37.8484&limit=50' \
  -H 'Content-Type: application/json;'
import requests

url = (
    "https://api.mining.skytruth.org/collections/public.annual_mining/items"
    "?bbox=-81.8156,37.7981,-81.7882,37.8484"
    "&limit=50"
)

data = requests.get(url).json()
print("Returned:", len(data.get("features", [])))

The guide’s example bbox falls in the Central Appalachian study area: -81.8156,37.7981,-81.7882,37.8484.

Query Model

Mining API requests are standard TiPG item queries. Most examples use a mix of bbox, direct property filters, and CQL-2 spatial predicates.

Parameter Type Usage
bbox minX,minY,maxX,maxY Primary spatial filter for examples in the guide.
limit integer Caps the number of returned features.
filter CQL-2 text Used for attribute filters like data_status NE 'provisional' and geometry predicates like s_intersects(geom,...).
id string Specific annual mining footprint identifier. Some IDs require URL encoding.
permit_id string Direct permit lookup on state permit collections.
mine_name string Facility name lookup on permit collections.
f string Output format, e.g. geojson or csv.

Filters

  • Bounding boxes use min_x,min_y,max_x,max_y.
  • Annual mine footprints can be filtered by data_status; the guide uses data_status NE 'provisional' to keep only cleaned yearly footprints.
  • Cross-dataset workflows use s_intersects(geom,<geometry>) in CQL-2 text filters.
  • Some mine IDs are not URL-safe. The guide explicitly encodes -303538+140420 before passing it as an id query.
  • Permit data is state-specific. Raw field consistency between public.state_permits_ky, _tn, _va, and _wv is not guaranteed. 
filter=data_status NE 'provisional'
filter=s_intersects(geom,<geometry>)
id=-303538%2B140420

Responses

Collection item endpoints typically return a GeoJSON FeatureCollection.

{
  "type": "FeatureCollection",
  "features": [
    {
      "type": "Feature",
      "id": "...",
      "geometry": { "...": "..." },
      "properties": { "...": "..." }
    }
  ]
}

public.annual_mining

Annual mine footprint polygons with year and data-cleaning status.

GET /collections/public.annual_mining/items
Field Description
geom Polygon geometry representing the mine footprint.
id Unique mine footprint ID.
mining_year Year in which the footprint was detected.
data_status final or provisional.
area Footprint area in square meters.
Open collection

public.state_permits_*

State permit datasets are published separately for Kentucky, Tennessee, Virginia, and West Virginia. The guide notes that attribute consistency is partial, not guaranteed.

Collection Coverage Notes
public.state_permits_ky Kentucky KyGovMaps, FOIA, and OSMRE GeoMine-derived fields.
public.state_permits_tn Tennessee OSMRE GeoMine plus FOIA bond data.
public.state_permits_va Virginia Virginia Department of Energy, OSMRE GeoMine, and FOIA merges.
public.state_permits_wv West Virginia WVDEP TAGIS, permit search, FOIA, and GeoMine-derived fields.

Common normalized fields used across the guide include permit_id, mine_name, permittee, mine_status, bond_status, avail_bond, full_bond, and surf_mine.

public.highwall_detections

Highwall detection geometries and cost/reclamation attributes. The guide marks this dataset as “coming soon” for some regions, so queries may legitimately return zero features depending on coverage.

GET /collections/public.highwall_detections/items
Field Description
highwall_id Unique highwall identifier.
rec_status Reclamation status from visual interpretation.
first_mined_yr / last_mined_yr Derived range for mining activity timing.
length / height Primary geometric measurements used in downstream estimates.
min_cost / mid_cost / max_cost Estimated reclamation cost range.
permit_id / mine_name Associated permit and mine metadata.

public.eamlis

OSMRE Abandoned Mine Land Inventory System problem areas hosted by SkyTruth. The source guide notes this hosted copy was last updated on 2025/10/24.

GET /collections/public.eamlis/items

Example fields include amlis_key, pa_name, county, watershed, mine_type, and ore_types.

public.counties

County boundaries filtered to the Central Appalachian study area. Data originates from the U.S. Census Bureau TIGER/Line shapefiles accessed on 2025/11/06.

GET /collections/public.counties/items

public.huc_boundaries

Watershed Boundary Dataset records filtered to the same study area. The guide documents a custom st_id field because some source records share the same objectid.

GET /collections/public.huc_boundaries/items

Example 2a: Return mine footprints that intersect a permit ID

This workflow fetches a permit geometry first, then intersects annual mining footprints while excluding provisional data. 

import requests
import urllib.parse
from shapely.geometry import shape

permit = requests.get(
    "https://api.mining.skytruth.org/collections/public.state_permits_wv/items?permit_id=S501889"
).json()["features"][0]

filter_text = (
    f"s_intersects(geom,{shape(permit['geometry']).wkt}) "
    "AND data_status NE 'provisional'"
)

mine_url = (
    "https://api.mining.skytruth.org/collections/public.annual_mining/items?filter="
    + urllib.parse.quote(filter_text)
)

mine_features = requests.get(mine_url).json()["features"]

Example 2b: Return mine footprints that intersect a mine name lookup

import requests
import urllib.parse
from shapely.geometry import shape

permit = requests.get(
    "https://api.mining.skytruth.org/collections/public.state_permits_wv/items?mine_name=HOBET 21 WEST RIDGE SURFACE"
).json()["features"][0]

filter_text = (
    f"s_intersects(geom,{shape(permit['geometry']).wkt}) "
    "AND data_status NE 'provisional'"
)

mine_url = (
    "https://api.mining.skytruth.org/collections/public.annual_mining/items?filter="
    + urllib.parse.quote(filter_text)
    + "&limit=100"
)

mine_features = requests.get(mine_url).json()["features"]

Example 2c: Return permits that intersect a specific mine footprint ID

import requests
import urllib.parse
from shapely.geometry import shape

mine_id = urllib.parse.quote("-303538+140420")
mine = requests.get(
    f"https://api.mining.skytruth.org/collections/public.annual_mining/items?id={mine_id}"
).json()["features"][0]

filter_text = f"s_intersects(geom,{shape(mine['geometry']).wkt})"
permit_url = (
    "https://api.mining.skytruth.org/collections/public.state_permits_wv/items?filter="
    + urllib.parse.quote(filter_text)
)

permit_features = requests.get(permit_url).json()["features"]

Coverage & Sources

SkyTruth datasets

The source guide lists annual mining footprints for 1985-2025 and a live highwall detections collection with evolving coverage.

Third-party reference layers

e-AMLIS was accessed on 2025/10/24. Watershed Boundary Dataset and Census TIGER county data were accessed on 2025/11/06 and then clipped to the Central Appalachian study area.

Permit data provenance

State permit boundaries were assembled from state mining databases, FOIA requests, and OSMRE GeoMine. Per-state raw fields differ.

The guide explicitly documents a custom st_id in public.huc_boundaries because some source objectid values are duplicated.
